# Nano Project – Game Store 

## Over dit project
Het doel van dit project is om de basiskennis van Python in de praktijk toe te passen, zodat we ervaring opdoen met het ontwikkelen van een werkende applicatie.
Dit project is ontwikkeld voor de opleiding HBO-ICT binnen het vak PROG. Het project bestaat uit een eenvoudige game store waarin twee spellen beschikbaar zijn:


- Raad het Nummer
- Galgje

## Functionaliteiten
- Keuzemenu om een game te selecteren
- Het spel Raad het Nummer, waarbij de gebruiker een getal moet raden
- Het spel Galgje, waarbij de gebruiker een woord moet raden letter voor letter
- Een functie om het weer te bekijken binnen het project



## Gebruik
Start het programma en kies een van de beschikbare games of bekijk het weer via het menu.

Gemaakt als onderdeel van een Nano project binnen HBO-ICT.
